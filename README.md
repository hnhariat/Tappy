# Tappy

#개요
탭으로 카운팅하는 애플리케이션

#개발목적
1. 단순한 터치만으로(한손만으로) 특정 동작을 완성시킬 수 있을까 고민
2. 슬라이드 동작으로 화면전환이 가능하게 구현

#추가개발사항
1. 이유가 분명해야하는 UX적 관점에서 슬라이드가 가능하다는 힌트 제공
